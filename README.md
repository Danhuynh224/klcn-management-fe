# Thesis Management FE

Frontend cho hệ thống quản lý đăng ký BCTT / KLTN, xây dựng bằng React, TypeScript, Vite và Ant Design.

## Công nghệ sử dụng

- React 19
- TypeScript
- Vite 8
- Ant Design
- React Router DOM
- TanStack Query
- Axios
- React Hook Form và Zod
- Zustand
- ESLint

Project được build bằng TypeScript compiler và Vite thông qua lệnh:

```bash
tsc -b && vite build
```

## Yêu cầu môi trường

- Node.js
- npm

## Cài đặt

```bash
npm install
```

## Cấu hình môi trường

Sao chép file `.env.sample` thành `.env` và cập nhật URL API nếu cần.

```bash
cp .env.sample .env
```

## Chạy dự án

```bash
npm run dev
```

Ứng dụng mặc định chạy trên địa chỉ Vite hiển thị trong terminal.

## Build production

```bash
npm run build
```
